CSV Upload Format

Please use the following CSV format when uploading metric data:

Required Format:
metricName,industry,country,year,value

Note: The first row will be treated as headers and skipped during upload.

Example:
GDP Growth Rate,ECONOMICS,US,2023,2.5
Unemployment Rate,ECONOMICS,UK,2023,4.2
Inflation Rate,ECONOMICS,CA,2023,3.1
